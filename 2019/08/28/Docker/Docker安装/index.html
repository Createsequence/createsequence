二. 设置阿里云镜像加速

由于国内访问DockerHub速度堪忧，所以需要使用国内的镜像，比如网易或者阿里云。

这里使用阿里云容器镜像服务，注册后开通服务，在主界面选择“镜像加速器”，再选择CenterOS即可。

1 . 根据提示，在配置文件中添加专属加速器地址配置

2. 重新加载配置文件，重启docker

注：网易云和阿里云操作基本相同，相对推荐使用阿里云

3. 确认配置生效
